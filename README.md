# Keypad 4x4

![KEYPAD_4X4](docs/keypad.png)

GPIO   |PIN | FUNC | Direction
-------|----|------|----------
16     | X1 | COL 0| Input
17     | X2 | COL 1| Input
13     | X3 | COL 2| Input       
19     | X4 | COL 3| Input
21     | Y1 | LIN 0| output
22     | Y2 | LIN 1| output
12     | Y3 | LIN 2| output 
25     | Y4 | LIN 3| output

# code --list-extensions | xargs -L 1 echo code --install-extension
